marathon01Info ↘
For a while I have been using a pen plotter to make experimental posters playing around with image and typography. The pen plotter can create interesting textures that can't be created with other printing techniques. You can use any pen with it, from thick black markers to dainty little glitter pens. I'm currently working on a little prothesis for it to make it fit larger pens and other tools.

I mostly create repeating patterns and waves that reveal bigger images, constantly trying to find new possibilities both with the machine and in the computer programs used to create a design.

Various plotted images, 2022-2023
